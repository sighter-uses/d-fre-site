D-Fre プライバシーポリシー 利用規約

プライバシーポリシー

最終更新日: 2026-05-25

D-Fre（以下「本アプリ」）は、ドイツ語学習者向けの単語学習アプリです。ユーザーのプライバシーを尊重し、以下のとおり個人情報を取り扱います。

1. 提供者

  • アプリ名: D-Fre
  • 提供者: Donexe
  • 連絡先: Sikun0708@gmail.com

2. 収集する情報

2.1 ユーザーが直接提供する情報

  • メールアドレス（メールアドレス登録時）
  • パスワード（ハッシュ化して保存）
  • ニックネーム（任意）
  • 学習設定（レベル、1日の目標、通知設定、表示設定）
  • マイ単語帳に登録した単語
  • 単語へのお気に入り・メモ

2.2 アプリ利用から自動的に発生する情報

  • 学習履歴（単語ごとの学習状態、正誤履歴、連続学習日数）
  • テスト結果（正答数、所要時間、誤答理由）
  • アプリ利用イベント（画面遷移、ボタンタップ等の匿名統計）※解析機能の導入後
  • クラッシュレポート（エラー内容、デバイス情報）※監視機能の導入後
  • 接続元 IP アドレス、User-Agent（セキュリティ目的、短期保持）

※ 現時点では、アプリ利用イベントの解析・クラッシュレポートの外部送信は行っていません。解析・監視機能は将来導入を予定しており、導入時に本ポリシーを更新します。

3. 利用目的

  • アカウントの認証・管理
  • 学習機能の提供（テスト出題、苦手復習、進捗表示）
  • 学習リマインダー通知
  • アプリの不具合修正・品質改善
  • 不正利用の防止

4. 第三者への提供

本アプリは以下の第三者サービスを利用しており、最小限のデータを提供します。「将来利用予定」とあるものは現時点では未導入で、導入時に本ポリシーを更新します。

サービス 提供者 用途 提供される情報
Supabase Supabase Inc.（米国） バックエンド（認証・DB・ストレージ） 第2項記載の全データ
Sentry Functional Software Inc.（米国） エラー監視（将来利用予定） クラッシュ情報、user_id、デバイス情報
PostHog PostHog Inc.（米国） プロダクト解析（将来利用予定） 画面遷移・イベント、user_id、デバイス情報（IP は丸めて保持）
Cloudflare Cloudflare Inc.（米国） 本サイト配信 標準的なアクセスログ
GitHub Releases GitHub Inc.（米国） Android APK 配信 ダウンロード時の IP・User-Agent

これら以外の第三者には、法令に基づく場合を除き提供しません。

解析・監視機能（Sentry / PostHog）は将来導入を予定しています。導入時には、設定画面から送信を停止（opt-out）できるようにします。

5. 国外移転について

上記第三者サービスはいずれも米国に主要設備があり、ユーザーデータは米国を含む海外で処理される可能性があります。EU 圏ユーザーの方は、GDPR に基づく権利を行使可能です（第7項参照）。

6. 保存期間

  • 認証情報・学習履歴: アカウント有効期間中
  • アカウント削除時: 即時削除（Supabase 上のユーザーデータは関連テーブルも含めて全削除）
  • Sentry / PostHog 上のデータ（将来導入時）: 各サービスの保持ポリシーに従う
  • バックアップ: 週次フルバックアップを 90 日保持

7. ユーザーの権利

ユーザーは以下を行使できます。

  • 開示: 設定画面で参照、または問い合わせ
  • 訂正: 設定画面でニックネーム・レベル等を変更
  • 削除: 設定 → アカウント削除で全データを消去
  • 利用停止: アカウント削除と同じ
  • 解析 opt-out: 解析・監視機能の導入後、設定画面で切替
  • データポータビリティ（エクスポート）: Phase 2 以降で提供予定。それまでは問い合わせ対応

EU/UK 圏のユーザーは GDPR、米カリフォルニア州ユーザーは CCPA、日本国内のユーザーは個人情報保護法に基づく権利を有します。

8. キャッシュ・ローカル保存

本アプリは端末内に以下を保存します（AsyncStorage / SecureStore / 端末キャッシュ）。

  • 学習履歴（オフライン学習のため）
  • 認証トークン（SecureStore）
  • 表示設定
  • 音声ファイル（オンデマンドキャッシュ）

これらのデータはアプリをアンインストール、または 設定 → データを消去 で削除できます。Web 版（ブラウザ利用）の場合、ブラウザのキャッシュをクリアすることで削除できます。

9. 子どものプライバシー

本アプリは 13 歳未満を主要対象としません。13 歳未満の方がご利用になる場合は、保護者の同意の上でご利用ください。

10. セキュリティ

  • 通信は HTTPS で暗号化
  • パスワードは bcrypt 等のハッシュ関数で保存（平文保持なし）
  • 認証トークンは端末の Secure Storage に保管
  • Row Level Security により、他ユーザーのデータには一切アクセスできません

11. ポリシー変更

本ポリシーは予告なく変更される場合があります。重要な変更があった場合はアプリ内通知または本サイトでお知らせします。
